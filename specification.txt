Class Css that I used in my homepage :
1. h1fahmi : style for h1 that contain setting for font-family, font size, color and text-align
2. centerfahmi : style for specific division that contain setting for font-family, font size, color and text-align
3. footerfahmi : style for footer that contain setting for position, left, bottom, width, height, background-color, color, text-align
4. h4fahmi : style for h4 that contain setting for font-family, font size, color and text-align
5. body : style for background color of Body
6. imgfahmi & imgfahmi2 : style for image that contain setting for width & height of image
7. video container & responsive video : style for video that contain setting for displaying 2 video in parallel
